# CV Application

A React-based CV/résumé builder. Users can add, edit, and submit personal information, education, and work experience.

## Features

- **General Information** — name, email, phone number
- **Educational Experience** — school name, field of study, dates
- **Practical Experience** — company, position, responsibilities, dates
- **Edit/Submit toggle** — switch between form and display mode for each section

## Built With

- React
- Vite

## Getting Started

```bash
npm install
npm run dev
```
